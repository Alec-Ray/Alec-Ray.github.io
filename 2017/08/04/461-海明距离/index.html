461.海明距离

2017-08-04

两个整数之间的汉明距离是对应的位在不同的位置数。

给出了两个整数和计算的汉明距离。xy The Hamming distance between two integers is the number of positions at which the corresponding bits are different. Given two integers x and y, calculate the Hamming distance. Note: 0 ≤ x, y < 231. Example:

Input: x = 1, y = 4

Output: 2

Explanation:
1 (0 0 0 1)
4 (0 1 0 0)
? ?

The above arrows point to positions where the corresponding bits are different.

解题方案：

既然是判断整数的二进制位上标识是否相等，那么我们可以先把整数从十进制数转换为二进制数，然后判断两个整数每一个对应二进制位的数值是否相等。十进制转化二进制的方法为：使用连续除2取余，将余数倒叙排列即可。

例如整数 5：

5 / 2 = 2—–1

2 / 2 = 1—–0

1 / 2 = 0—–1

5 => 101

由十进制转换二进制的过程可知，我们直接在进制转换的过程中两整数二进制位即可，直到最大的整数除尽。

class Solution
{
public:
int hammingDistance(int x, int y)
{

    int count=0;

    while(x!=0||y!=0)
    {
        int m=x%2;
        int n=y%2;

        if(m!=n)
        {
            count++;
        }

        x/=2;
        y/=2;
    }

    return count;

}

};
