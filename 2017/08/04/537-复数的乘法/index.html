537.复数的乘法.

2017-08-04

Given two strings representing two complex numbers. You need to return a string representing their multiplication. Note i2 = -1 according to the definition.

给定两个字符串表示两个复数。

您需要返回一个字符串，表示它们的繁衍。请注意我2=-1 的定义。 Example 1:

Input: “1+1i”, “1+1i”
Output: “0+2i”
Explanation: (1 + i) * (1 + i) = 1 + i2 + 2 * i = 2i, and you need convert it to the form of 0+2i.

Example 2:

Input: “1+-1i”, “1+-1i”
Output: “0+-2i”
Explanation: (1 - i) * (1 - i) = 1 + i2 - 2 * i = -2i, and you need convert it to the form of 0+-2i.

Note:

  1. The input strings will not have extra blank.
  2. The input strings will be given in the form of a+bi, where the integer a and b will both belong to the range of [-100, 100]. And the output should be also in this form.

注意︰

1.输入的字符串不会有额外的空白。

2.A + bi，形式的整数和b将两者都属于地方的范围时，将给予输入的字符串 [-100，100]。和的输出应该也在此窗体。

class Solution
{
public:
string complexNumberMultiply(string a, string b)
{
int ra,ia,rb,ib;
char buff;

    stringstream aa(a),bb(b),ans;

    aa>>ra>>buff>>ia>>buff;
    bb>>rb>>buff>>ib>>buff;

    ans<<ra\*rb-ia\*ib<<"+"<<ra\*ib+rb\*ia<<"i";

    return ans.str();
}

};

高中数学复数公式：

两个复数的乘积：(a+bi)(c+di)=(ac－bd)+(bc+ad)i.

加法结合律： (a+bi)+(c+di)=(a+c)+(b+d)i.

结合律： z1+z2=z2+z1; (z1+z2)+z3=z1+(z2+z3).

共轭复数：a+bi和a-bi

复数的模z=a+bi，∣z∣=√（a^2+b^2)
