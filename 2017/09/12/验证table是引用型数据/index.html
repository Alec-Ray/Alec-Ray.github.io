验证table是引用型数据

2017-09-12

代码如下：

str01=”wjgbaby.com”
str02=”wjgbaby.com”

table01={“111”,”222”,”333”}
table02={“111”,”222”,”333”}

print(str01==str02)
print(table01==table02)

结果如下：

>lua -e “io.stdout:setvbuf ‘no’” “4.lua”
true
false

Exit code: 0

因为str01和str02里面数据是值类型，因为值相同，所以返回为true table01和table02虽然表里的成员相同，但是存储地址不同，即为引用类型，所以返回false 总结： 值类型的数据判断相同的条件是：值（内容）相同； 引用型的数据判断相同的条件是：地址相同.
