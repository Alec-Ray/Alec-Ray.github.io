1022. D进制的A+B (20)

2017-11-11

输入两个非负10进制整数A和B(<=230-1)，输出A+B的D (1 < D <= 10)进制数。 输入格式： 输入在一行中依次给出3个整数A、B和D。 输出格式： 输出A+B的D进制数。 输入样例：

123 456 8

输出样例：

1103

代码如下：

#include
using namespace std;

void transform(int m, int k)
{
if (m / k == 0)cout << m;
else
{
transform(m / k, k);
cout << m%k;
}
}
int main()
{
int A, B, D;
cin >> A >> B >> D;
int sum = 0;
sum = A + B;
transform(sum, D);
return 0;
}
