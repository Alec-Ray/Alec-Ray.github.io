一.Scrollbar 介绍

Scrollbar：卷动条。浏览器最右边竖向的下滑条就是卷动条的例子。

二.Scrollbar 组成部分分析

Scrollbar Image [整体功能控制+ 背景图片显示] |—空物体[控制子物体活动范围] |—Image [滑块图片] [

三.Scrollbar 组件特有属性

Size：用于控制滑块的大小； Number Of Steps：用于控制滑块每次移动的步长。

四.Scrollbar 事件监听

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;

public class tempUGUI : MonoBehaviour {

private Scrollbar m\_Scrollbar;

void Start () {
    m\_Scrollbar = gameObject.GetComponent<Scrollbar>();
    m\_Scrollbar.onValueChanged.AddListener(ButtonClickTest);
}

private void ButtonClickTest(float value){
    Debug.Log(value);
}
